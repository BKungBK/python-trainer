# Python Trainer

Python Trainer คือแอปพลิเคชันสำหรับฝึกเขียนโค้ดภาษา Python ที่ทำขึ้นมาเพื่อให้การฝึกทำโจทย์ในทุกๆ วันเป็นเรื่องง่ายและสนุกขึ้น ตัวแอปถูกออกแบบมาให้เบา ทำงานได้รวดเร็วบนเครื่องคอมพิวเตอร์ของคุณเอง และมีฟีเจอร์สำหรับพัฒนาทักษะไปพร้อมๆ กับเพื่อนได้

## ทำอะไรได้บ้าง

* **เป้าหมายประจำวัน (Daily Challenges):** แอปจะทำการสุ่มโจทย์ตามหมวดหมู่ต่างๆ มาให้เราได้ฝึกฝนตามโควตาเป้าหมายที่ตั้งไว้ในแต่ละวัน เพื่อสร้างวินัยการเขียนโค้ดอย่างสม่ำเสมอ
* **พื้นที่พิมพ์โค้ดและตรวจผลทันที (Code Editor & Testing):** มีหน้าจอพิมพ์โค้ด (Monaco Editor) ในตัว สามารถกดรันเพื่อทดสอบผลลัพธ์ของโค้ดกับเคสตัวอย่าง หรือกดส่งคำตอบเพื่อตรวจผ่านเคสทดสอบจริงได้ทันที
* **ดูความคืบหน้าของเพื่อน (Friend Activity):** มีระบบซิงค์ข้อมูลกับคลาวด์เพื่อให้เราเห็นว่าเพื่อนร่วมเรียนรู้ของเรากำลังแก้โจทย์ข้อไหนอยู่ ออนไลน์อยู่หรือไม่ หรือทำโจทย์เสร็จครบตามเป้าหมายของวันหรือยัง
* **ใช้งานแบบออฟไลน์ได้:** ถึงแม้จะไม่มีอินเทอร์เน็ต แอปก็ยังสามารถบันทึกประวัติการส่งโค้ดและฝึกฝนโจทย์เดิมในเครื่องต่อได้ และจะซิงค์ข้อมูลขึ้นคลาวด์ให้เองเมื่อระบบกลับมาออนไลน์

---

## วิธีใช้งานเบื้องต้นสำหรับผู้ใช้ทั่วไป

1. **ดาวน์โหลดและติดตั้ง:** ไปที่แท็บ Releases บนหน้า GitHub นี้ ดาวน์โหลดไฟล์ติดตั้งล่าสุด (เช่น ไฟล์ที่มีนามสกุล .msi หรือ .exe) แล้วกดติดตั้งลงเครื่องคอมพิวเตอร์ของคุณ
2. **ตรวจสอบ Python ในเครื่อง:** แอปนี้จำเป็นต้องใช้ระบบรัน Python ของเครื่องคุณในการประเมินผลโค้ด กรุณาติดตั้ง Python 3 ลงในเครื่องคอมพิวเตอร์ของคุณก่อนเปิดใช้งาน
3. **เปิดแอปและตั้งค่า:** 
   * เมื่อเปิดแอปขึ้นมา ให้ไปที่หน้า "ตั้งค่า" (Settings)
   * กดปุ่ม "ค้นหาอัตโนมัติ" เพื่อให้แอปเชื่อมต่อกับ Python ในเครื่องของคุณ
   * เลือกชื่อโปรไฟล์ผู้ใช้งานของคุณ
   * กดปุ่ม "ซิงค์ข้อมูลทันที" เพื่อดึงโจทย์ล่าสุดลงมาเก็บไว้ในเครื่อง
4. **เริ่มฝึกฝน:** กลับไปที่หน้าหลัก แล้วกดเริ่มทำโจทย์ท้าทายประจำวันได้เลย!
